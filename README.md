## Use

***

### What do you need to start

1. Package manager [NPM](https://www.npmjs.com/).
2. Some CLI to execute commands from directory of your project.
3. `optional`: VS Code editor.

### Installation

Just clone this repository and execute:

```bash
npm i
```
